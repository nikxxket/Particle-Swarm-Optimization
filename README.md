# Визуализация расстановки сенсоров методом роя частиц (PSO)

Данный проект демонстрирует применение алгоритма **роя частиц (Particle Swarm Optimization, PSO)** для оптимального размещения сенсоров на двумерной плоскости. Цель – максимизировать покрытие заданной области при ограниченном количестве сенсоров и с учётом штрафа за перекрытие их зон действия.

## Описание алгоритма

- **Пространство поиска** – координаты центров всех сенсоров (12 сенсоров).
- **Целевая функция** – процент покрытых пикселей поля (шаг дискретизации 3 пикселя) минус штраф за перекрытие зон покрытия (чем больше сенсоров покрывают один пиксель, тем выше штраф). Значение ограничено [0, 100].
- **Параметры PSO**:
  - Количество частиц: 40
  - Количество итераций: 300
  - Инерционный вес `w = 0.7`
  - Когнитивный коэффициент `c1 = 1.5`
  - Социальный коэффициент `c2 = 1.5`
  - Максимальная скорость: 20 пикселей на итерацию
- **Границы** для центров сенсоров расширены на радиус сенсора, чтобы круги могли полностью покрывать крайние области.

## Визуализация

- **Основное поле** – сетка 900×700 пикселей.
- **Тепловая карта покрытия**:
  - Зелёный – покрытие одним сенсором
  - Жёлтый – перекрытие двух сенсоров
  - Оранжевый – перекрытие трёх и более сенсоров
- **Частицы** – синие точки (промежуточные решения)
- **Лучшее решение** – красные круги (радиус 100 пикселей) с номерами сенсоров
- **График сходимости** – отображает прогресс покрытия в реальном времени
- **История лучших позиций** – полупрозрачные круги, показывающие эволюцию лучшего решения

## Управление и обратная связь

После запуска приложение автоматически выполняет 300 итераций PSO. На экране отображаются:
- текущая итерация и достигнутый процент покрытия;
- прогресс-бар;
- график изменения покрытия по итерациям;
- легенда цветов.

В консоль каждые 50 итераций выводится промежуточный результат.
